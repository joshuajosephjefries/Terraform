DOCKER

Terraform is an IaC software that we used to use to write, plan and apply the code for configuration and deployment of applications. You can deploy, release, scale, and monitor the applications in real time across multiple cloud providers. Developed by Hashicorp Inc, used to develop config files to create, manage and destroy resources or infrastructure. Provision and manage all the infrastructure throughout it's lifecycle. Manages low level components like compute, storage and networking resources. Also Manages high level components like DNS entries and SaaS features. Terraform works through the APIs. Providers enables to work with any platform using an API.

COMMANDS

terraform init: This command initialises terraform and downloads all the required plugins and modules

terraform plan: This command plans the infrastructure and shows the blueprint ahead of time before execution
terraform apply: This command applies all the changes and builds infrastructure on cloud
